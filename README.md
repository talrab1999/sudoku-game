A sudoku game i've built as the final project fom my C course in college.
In this sudoku game you can choose at the start of the game the number of players, and choose the name of each player.
The game chooses the square for the player with an algorithm that checks which square has the least mount of possibilities to choose from, and presents the possibilities to the player.
The game is played as in each round each player gets to play 1 turn.
The first one to take a turn is the one who got the least squares opened at the start of the game, and the last one is vice versa.
